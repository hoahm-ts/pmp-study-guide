# PMP Study Guide

A comprehensive, bilingual (English / Vietnamese) PMP exam preparation guide built around the **PMI Exam Content Outline (ECO) 2021**.

Covers all 3 domains and 35 tasks with real-world fintech scenarios, practice questions, professional templates, and AI prompt toolkits for modern project managers.

### Each lesson includes
- **Bilingual content** (EN/VI) — core concepts explained in both languages
- **Real-world scenario** — fintech context (digital lending platform)
- **3 practice questions** — PMP exam style with full answer explanations
- **Professional templates** — Jira / Confluence / process templates ready to use at work
- **AI Tools section** — sample Claude prompts, Jira/Confluence templates, and process integrations for AI-augmented PM workflows

### Additional reference pages
- PMBOK 7 — 12 Principles & 8 Performance Domains
- Tools & Templates — EVM formulas, Thomas-Kilmann, Tuckman, Hersey-Blanchard, contract types, agile ceremonies
- Exam Tips — brain dump template, time strategy, question patterns
- PM Mindset — servant leadership, ECO philosophy
